win10系统安装RN开发环境

By lanveer; Published on 2019-07-01

  • React Native

记录win10系统下安装react-native 开发环境

  本着学习的原则，同时也满足一下自己的好奇心，虽然不会原生的开发，但是对于app那是热情不减啊，现在就有这么一个跨平台的东西放在这里，所以自己也来尝个鲜，虽然这玩意出来挺长一段时间了，由于之前自己的懒惰，也没有去学习，亡羊补牢，为之不晚，下面就开始各种安装之旅吧。

注意事项：
1.  一部安卓手机，（最好是安卓6以上，需要打开开发者模式， 打开USB调试，方便电脑链接，测试方式可以使用adb devices, 可以查看电脑链接的手机数量，，如果没有就在Android studio中用模拟器代替，稍后会提到）

2.最好是能够科学上网，毕竟很多的包是需要从境外服务器下载的，到时候需要配置一些东西

3.win10系统电脑（最好是Intel处理器，因为后面模拟器安装的时候会遇到）

4.配置好node开发环境，使用npm安装 npm install -g yarn react-native-cli （项目脚手架生成工具）
操作步骤：
1.下载jdk（注意， 最新的jdk运行方式和RN 生成项目的gradle有一些冲突和不兼容， 需要做额外的处理）

2. 配置JAVA_HOME，指向jdk安装的bin目录 1.2. 配置path目录（因为打包编译的时候会用到，找cmd中输入java-version的时候能够正常输出为准） 1.3. 配置ANDROID_HOME（这个指向android studio 的默认sdk目录）

3.下载android studio并安装（安装的时候最好是科学上网中，以customer 的方式安装， 注意勾选Android SDK Android SDK Platform Performance (Intel ® HAXM) Android Virtual Device）

4.安装sdk,在SDK Manager 下面配置 （默认路径为c:\Users\你的用户名\AppData\Local\Android\Sdk） Android SDK Platform 28 Intel x86 Atom_64 System Image，   最后这些sdk安装之后，需要在配置ANDROID_HOME 的时候指向这里）

5.. 把 platform-tools 目录添加到环境变量 Path 中（默认路径为c:\Users\你的用户名\AppData\Local\Android\Sdk\platform-tools）

6.下载python（注意一定是2.xx版本，因为3以上的版本不兼容）

7.安装项目管理工具git（这个可有可无，可以忽略）

8.配置模拟器，这里暂时不讲，直接在IDE中，根据步骤就可以配置了，然后通过adb-devices便可以测试

9.运行demo环境， 进入项目目录， 然后执行， react-native run-android
